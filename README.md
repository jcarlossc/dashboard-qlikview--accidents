# 📌 ANÁLISE DE ACIDENTES DE TRÂNSITO – RECIFE / 2019
Dashboard Estratégico de Mobilidade Urbana

---

## 📌 Objetivo do Projeto
Este projeto tem como objetivo analisar e visualizar dados de acidentes de trânsito, permitindo a identificação de padrões, tendências e 
indicadores-chave relacionados à ocorrência de acidentes.

Por meio de dashboards interativos desenvolvidos no QlikView, o projeto busca apoiar a tomada de decisão, fornecendo informações claras sobre:
* Evolução dos acidentes ao longo do tempo
* Distribuição por tipo de acidente
* Gravidade (com vítimas / sem vítimas)
* Principais fatores associados às ocorrências
* Entre outros

O foco é transformar dados brutos em informação estratégica, de fácil compreensão para diferentes perfis de usuários.

---

## 📌 Público-Alvo

O dashboard foi desenvolvido para atender principalmente:

* Gestores públicos e analistas de trânsito
* Tomadores de decisão em órgãos governamentais
* Analistas de dados e BI
* Pesquisadores e estudantes da área de mobilidade urbana
* Cidadãos interessados em compreender dados de segurança viária

A interface prioriza clareza visual, navegação intuitiva e autonomia do usuário, mesmo para quem não possui conhecimento técnico em análise de dados.
